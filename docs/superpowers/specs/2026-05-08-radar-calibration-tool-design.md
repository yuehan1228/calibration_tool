# 雷达自动标定工具设计文档

## 概述

Web 界面的雷达标定工具，支持激光雷达和毫米波雷达的单次标定。用户上传坐标文件、输入大机姿态参数，自动计算标定矩阵和误差。

## 功能需求

### 输入
- **坐标文件上传**：GNSS 天线坐标 + 雷达坐标，支持 CSV 和 TXT 格式
- **传感器类型选择**：激光雷达 / 毫米波雷达
- **目标坐标系选择**：
  - 旋转中心坐标系（默认）- 无额外参数
  - 臂架坐标系 - 需要臂架长度、伸缩距离
  - 溜筒坐标系 - 需要臂架长度、伸缩距离、连接点距离
- **大机姿态参数**：
  - 回转角（slewing_angle）
  - 俯仰角（luffing_angle）
  - 行走距离（walking_dist）
  - 旋转中心坐标（rot_center: x, y, z）
  - 伸缩距离（stretch_dist）
  - 臂架长度（arm_length，默认 25.3）
  - 连接点距离（arm_to_connect_point_dist，仅溜筒坐标系）

### 输出
- 标定矩阵 T_fixed（4x4 齐次变换矩阵）
- 变换误差（每个点的欧氏距离误差）

## 技术方案

### 架构
- **后端**：Flask 轻量级 Web 框架
- **前端**：HTML + CSS + JavaScript（原生，无框架）
- **计算核心**：复用现有代码的坐标变换和 SVD 求解逻辑

### 核心模块

#### 1. 文件解析模块 (parser.py)
- 解析 CSV 和 TXT 文件
- 提取 x, y, z 坐标列
- 返回 numpy 数组

#### 2. 坐标变换模块 (transform.py)
- `rodrigues()`: 罗德里格斯旋转公式
- `transform_to_rot_center()`: 变换到旋转中心坐标系
- `transform_to_arm()`: 变换到臂架坐标系
- `transform_to_chute()`: 变换到溜筒坐标系

#### 3. 求解模块 (solver.py)
- `estimate_rigid_transform()`: SVD 方法求解刚体变换
- `compute_error()`: 计算变换误差

#### 4. Web 应用 (app.py)
- `/`: 主页面
- `/calculate`: POST 接口，接收参数并返回计算结果
